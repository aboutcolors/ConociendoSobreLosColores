Tendencia y Color


Poderosos

El color rojo no puede ser ignorado. Es el color "poder" fundamental. Se usa para transmitir sentimientos fuertes, vitalidad.

Pruébame

Amistosos

El naranja, junto con sus vecinos, se utilizan se usa para transmitir energia, vitalidad, amistad.

Pruébame

Acogedores

Combinaciones del naranja amarillento, ambar, son colores acogedores. Si combinamos esto, con un toque de rojo, se crean tonos radiantes.

Pruébame

Orientadores

Uso de verdes con una agradable combinación chocante , no hay un esquema establecido o un color fijo, pero si queremos definir, uso de verdes claros define esta moda.

Pruébame

Refrescantes

Verde azulado, unido al naranja rojizo, o a un verde oscuro. Usado para describir viajes y tiempo libre.

Pruébame

Profesionales

El término profesional remite a colores negros, grisáceos y diferentes tonos de estos colores hasta llegar al blanco, estos por falta de características personales y ser mas neutros.
